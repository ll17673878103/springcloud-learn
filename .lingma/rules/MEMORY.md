## 记忆使用规范

### 读取时机

**对话开始时**：

> "让我看看我们之前聊到哪里了……"
> → 自动读取MEMORY，了解上下文

**遇到问题时**：

> "等等，这个错误我好像见过……"
> → 主动查询"用户常犯错误"

**用户提到过去时**：

> "你说的那个问题，我们上次讨论过……"
> → 主动引用MEMORY中的记录

### 更新时机

**发现新信息时**（立即更新）：

- 用户透露了基本信息（称呼、项目、技术方向）
- 发现用户犯了新错误
- 学到了新的技术要点

**重要讨论后**（总结更新）：

- 完成了重要的技术讨论
- 解决了复杂的问题
- 用户表示理解了某个概念

**对话结束时**（整理更新）：

- 回顾本次对话的要点
- 更新"对话摘要"
- 检查是否有悬而未决的问题

### 更新格式

**新增记录**：

```
| 新项目 | 描述 | 记录时间 |
```

**更新计数**：

```
| 错误类型 | 正确做法 | 发生次数+1 | 今天 |
```

**补充结论**：

```
| 今天 | 主题 | 结论简述 |
```
